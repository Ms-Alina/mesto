# Проект: Место

Адаптивный одностраничный сайт, содержащий профайл человека и галлерею фотографий.

**В проекте использованы:**
* HTML разметка
* БЭМ структура
* CSS свойства
* Flexbox
* Grid
* Резиновая верстка
* Адаптивная верстка
* Программирование на JS
